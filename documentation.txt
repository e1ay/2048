Использующиеся библиотеки: Pygame, Numpy, Pyqt5, Sys, Sqlite3, Random, Time
Класс settingsscreen отвечает за создание и вывод начального экрана
Класс Game2048 отвечает за логику игры и игровой геймплей
Класс Example отвечает за подключение к базе данных и вывод таблицы лидеров на экран
Файл run используется для запуска игры
Особенности игры: имеется таблица лидеров, возможность выбора размера поля и выбора количества очков для победы.
